#### Performance(성능) 이해하기
* Algorithm
	* 실행될 연산 갯수를 결정한다.
* Programming language, compiler, architecture
	* 실행될 기계 명령어 갯수를 결정한다.
* Processor and memory system
	* 명령어 실행 속도에 영향을 준다.
* I/O system (including OS)
	* I/O 시스템은 입출력 작업을 관리한다.
	* 운영체제는 I/O 작업을 제어하고 최적화한다.
	* 전체 시스템 성능에 영향을 준다.
